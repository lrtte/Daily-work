“没有人是与世隔绝的孤岛”。面对来势汹汹的新冠肺炎疫情，一批批战“疫”勇士们舍生忘死、冲锋陷阵，用生命守护生命。人心齐，泰山移。一个个平凡的劳动者守望相助、各司其职，团结成为战胜疫情的命运共同体。疫情笼罩之下，每一个感人的故事，每一秒感人的瞬间，无不迸发出振奋人心的精神力量。习近平总书记指出：“中国人民在疫情防控中展现的中国力量、中国精神、中国效率，展现的负责任大国形象，得到国际社会高度赞誉。”这场抗疫中淬炼出的精神，将在中华民族精神谱系上写下浓墨重彩的一笔。

　　

舍生忘死 勇毅笃行

　　

抗疫如战。大战来临，有人选择离开，更有人选择勇敢逆行。据国家信息中心大数据显示，在未统计港澳台地区的情况下，武汉“封城”的第一周内，就有12.38万人次从湖北省以外的其他地方进入武汉。他们中有救死扶伤的医生和防疫专家，有保物流生命线的司机，有除夕夜集结驰援的解放军……这些向疫区方向前行的人们，被称为新时代“最美逆行者”。 　　“常思奋不顾身，而殉国家之急”。面对生死，战斗在最前沿的广大医护人员勇毅笃行，挺立在疫情的风口，为病人撑起了生命的蓝天。无数人在党和人民最需要的地方冲锋陷阵、顽强拼搏，在平凡的工作岗位上创造不平凡的成就，展现了无惧风雨的英雄本色。 　　一位名叫张敏的安徽护士，忍住眼泪放下怀中年幼的女儿，转身奔赴武汉，临走时她说，“妈妈要去打怪兽了，很快就回来”。一位名叫骆名良的火神山医院装修工人，刚领到施工单位发放的7200元工钱加300元补助，就马上买了100多箱牛奶，送给附近同济医院的医务工作者和执勤人员。一位名叫汪勇的武汉快递小哥，在疫情期间自发接送上下班的医务人员，并送上热腾腾的盒饭。丹心寸意，皆为有情。在这场战“疫”中，无数人以担当之勇、奋斗之志，在挑战中挺起不屈的脊梁，谱写出一曲曲荡气回肠的华彩乐章。他们的舍生忘死、英勇奋战，正是中华民族不屈不挠、顽强奋斗的精神写照。

　　

守望相助 众志成城

　　

阻击疫情的前线是战场，提供保障的后方同样是战场。在维护社会基本运转的链条上，在疫情防控阻击战的防线上，快递员、保洁员、供电员、志愿者、环卫工等很多群体都在承担重任、默默付出。正如一句网络流行语所说，“没有天生的英雄，有的不过是挺身而出的普通人”。这些基层劳动者的奋力担当，是中国战“疫”经验的重要组成。 　　疫情像是一面镜子，以一种特殊方式，映照出普通人的耀眼光芒。疫情发生后，我们身边不乏心系武汉的“匿名捐款”者，给医护人员寄去“匿名礼物”的普通市民，为民警留下“匿名红包”的热心人……无数普通的社会成员，以自己的方式发光发热，汇聚成一股暖流。 　　在疫情防控中，中国之所以能取得阶段性重要成效，联防联控、群防群控防控体系发挥了重要作用。在社区，党员上一线、打头阵，分片包干，落实属地防控、联防联控、群防群控任务，工作人员加强出入卡点值守，用大数据、网格化浇筑疫情防控的坚强堡垒；在车间，医疗企业开启节假日不休模式，连续赶工，源源不断地生产出一批批急需的医用口罩、防护服；在家庭，人人加强防护，疏导心理，互相打气，增强信心，缓解焦虑。 　　“积力之所举，则无不胜也”。同舟共济、守望相助的精神力量，来自亿万人民聚沙成塔、握指成拳的爱国之志，彰显了中华民族风雨同舟、患难与共的家国情怀。

　　

命运与共 携手应对

　　

单则易折，众则难摧。这次疫情，让人们更清晰地认识到“人类命运共同体”的深刻含义。

　　

开放合作、守望相助，命运与共、风雨同担，这是人类历史中淬炼出的精神之光。中国人民抗击疫情，从一开始就秉持着人类命运共同体理念。疫情发生后，中国及时向包括世界卫生组织、有关国家和地区性组织在内的国际社会通报疫情信息，以前所未有的速度甄别出病原体，第一时间同世卫组织和其他国家分享病毒基因序列，为研制出病毒快速检测工具提供了可能，并同世界各国开展抗疫经验分享和交流。中国抗疫，不仅是对中国人民生命安全和身体健康负责，也是对全球公共卫生事业尽责。 　　滴水之恩当涌泉相报，是中华民族的传统美德。中国人民不会忘记，在中国疫情防控的艰难时刻，“山川异域，风月同天”的温暖，“中国加油”“武汉加油”的呐喊，国外友人捐款捐物的支援，世界各地点亮“中国红”的祈愿。而当其他一些国家和地区遭受疫情冲击时，中国人民也感同身受、施以援手。从中国专家团队奔赴国外抗击疫情前线画出一道道“最美逆行之弧”，到向亚洲、欧洲、非洲多国提供检测试剂盒，再到尽一切努力为其他国家采购医疗防护物资提供方便，中国以各种各样的方式，向其他出现疫情扩散的国家和地区提供力所能及的援助。中国人民与世界人民并肩抗疫、携手前行，让人类的文明之光熠熠生辉。 　　前不久，德国海德堡大学医院的托马斯·拉贝大夫求援，需要中国生产的一组药品，除了救他自己和家人，也想救他的患者。在得知该医生是被称为“中国辛德勒”的约翰·拉贝的孙子时，一家浙江药企立刻决定免费赠药。网友们纷纷表示：“这个忙咱中国人必须帮！” 　　爱之花绽放的地方，一切欣欣向荣。灾难面前，人性的光辉如同寒冬里的火炬，总是给人以温暖和力量。病魔是人类共同的敌人，全世界是一个命运共同体。我们唯有同舟共济、携手并进，才能战胜疫情、共渡难关。（经济日报-中国经济网评论员 梁瑜）
